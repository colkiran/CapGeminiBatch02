Dictionary
----------
1. keys in a dictionary is unique
2. unordered collection - 
 {'phy': 87, 'che': 96, 'bio': 99} - {'bio': 99, 'phy': 87, 'che': 96}
 (before python 3.7)

3. It is mutable
4. dictionaries are enclosed in { }
